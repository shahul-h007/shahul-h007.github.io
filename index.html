🎮 Devverse
🎮

Devverse is a mobile game app where players build their own game studio, grow systems over time, unlock upgrades, and progress through an idle management loop built around momentum and strategy.


I built it as a solo Flutter project, handling the full app flow from UI structure and gameplay systems to persistent saves, authentication, release preparation, and mobile-first polish.


The app includes login and registration with Supabase, local progress saving, settings and audio controls, trading and progression systems, and Android release preparation with monetization-ready AdMob integration.

📦 Borrow Box
📦

Borrow Box is a peer-to-peer lending and borrowing platform — the idea being that you shouldn't have to buy something you'll use once or twice when someone nearby already owns it.


The mobile app is built with Flutter, the backend runs on Python (Django), the database is SQL, and there's a web-facing shop view built with HTML/CSS. Users can list items, browse nearby listings, send borrow requests, and rate each other after a transaction.


This is my completed BCA graduation project — built end-to-end from design to deployment.
